A team gets two codes: one for the Transmitter (sees the map) and one for the Receiver (controls the boat). Hand each code to the right side.

Quick Help

  • ▸ Each game uses a fresh random map.
  • ▸ Score = seconds + (damage × 2). Lower is better.
  • ▸ Boat starts with 100 HP. Each mine = 25 damage.
  • ▸ Use Reset to give a team a new map.
  • ▸ Delete removes the team entirely.
